# Лабораторна робота 1 

Невеликий код, в якому програма приймає цілі числа та виводить їх суму в термінал.

## Як запустити
Запускаємо в терміналі код, наприклад щоб порахувати суму чисел 5 та 6:
```
python prog.py 5 6
```
Має видати: 
'11'
## Пояснення коду
 
`import argparse` — підключаємо бібліотеку для читання аргументів з командного рядка

`parser = argparse.ArgumentParser(description="Sum integers")` — створюємо парсер, який буде читати що користувач пише після назви програми

`parser.add_argument("numbers", nargs="*", type=int)` — задаємо парсеру що очікуємо числа. `nargs="*"` означає будь-яка кількість чисел, `type=int` означає що це цілі числа

`args = parser.parse_args()` — запускаємо парсер. Після цього `args.numbers` містить список чисел які ввів користувач
Наприклад `prog.py 5 6` дає `args.numbers = [5, 6]`.

`print(sum(args.numbers))` — `sum()` рахує суму списку чисел, `print()` виводить результат на екран
