# BlockProcessor with SQL backend

## Структура проєкту

Цей репозиторій містить лабораторні роботи 2-6, які реалізують систему BlockProcessor з SQL-бекендом:

| Лаба | Опис |
|:-----|:-----|
| **Lab 2** | Початкове налаштування BlockProcessor та обробка даних |
| **Lab 3** | Інтеграція SQL-бекенду та розробка схеми бази даних |
| **Lab 4** | Обробка запитів та логіка взаємодії з БД |
| **Lab 5** | Оптимізація роботи та обробка виняткових ситуацій |
| **Lab 6** | Фінальна реалізація, валідація моделей та тестування |

## Архітектура системи

Візуалізація процесу обробки даних у проєкті за допомогою Mermaid:

```mermaid

    A[Старт] --> B[Отримання блоку]
